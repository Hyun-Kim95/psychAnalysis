# 릴리즈 노트 (스타일) — 2026-04-10

## 요약

Figma `Page 3` 시안과 맞추기 위해 프론트엔드 글로벌 스타일을 CSS 변수(`--pa-*`)로 재정의하고, Manrope / Public Sans 웹폰트를 로드했습니다. Primary 색은 `#00288e`, 페이지 배경은 `#f7f9fb` 계열로 통일했습니다.

## 사용자 영향

- 검사 목록 카드·버튼·진행 바·결과 화면의 색과 글꼴이 시안에 더 가깝게 보입니다.
- 레이아웃 문구·기능 흐름은 변경하지 않았습니다.
